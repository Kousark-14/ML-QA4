# Clustering Implementation and Visualization

## Objective

To implement and compare K-Means and DBSCAN clustering algorithms.

## Files

* Clustering.ipynb

## Algorithms Used

* K-Means

## Results

Both algorithms successfully identified clusters in the dataset. DBSCAN handled noise better, while K-Means produced clear cluster separation.

## Conclusion

K-Means and DBSCAN were implemented and visualized successfully using Python.
